Dato un array di oggetti rappresentante un team di un’azienda, creare una pagina dedicata  in cui mostrare una card per ciascun componente.

Partite con il minimo HTML e CSS necessari.

Trovate l’array del team all’interno della cartella in allegato.

Bonus
Rendere l’esercizio responsive, mandando a capo le card

Bonus #2
Creare un pulsante che aggiunge un nuovo oggetto nell'array con i vostri dati.
Quando il pulsante viene premuto, effettuata di nuovo tutta la logica col ciclo che stampa le card in pagina
